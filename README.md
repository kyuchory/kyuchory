### 🦢 Introduction
- 📱 **React Native & React**를 활용해 웹과 모바일 앱을 아우르는 일관된 사용자 경험을 설계합니다.
- ⚡ **성능 최적화**에 진심입니다. 불필요한 리렌더링을 방지하기 위해 **React Hooks(useMemo, useCallback)**와 다양한 최적화 옵션을 전략적으로 활용하여 사용자 인터페이스의 반응성을 높입니다.
- 🟢 **Node.js** 기반의 서버 로직과 **MySQL**을 이용한 효율적인 **스키마 설계 및 쿼리 최적화**를 통해 전체 시스템의 퍼포먼스를 관리합니다.
- 단순 구현을 넘어, 서비스의 확장성과 유지보수성을 고려한 클린 코드를 지향합니다.

### 🗂️ Major Projects
- **개인 프로젝트**
    - `애니멀톡(Animal Talk)`: **React Native** 기반 반려동물 커뮤니티 앱 🐾 (성능 최적화 적용)
    - `메이플스토리 정보 검색`: API 데이터 가공 및 렌더링 성능 최적화를 통한 대시보드 구현
    - `URL 단축 서비스`: **Node.js** 기반 단축 로직 및 DB 성능 최적화

    - `여행 정보 서비스`: 대용량 데이터의 효율적인 시각화 구현
